As principais consultas em bancos de dados, existem alguns tipos comuns de consultas que são frequentemente usados:

1. SELECT:
 é a consulta mais básica em bancos de dados, usada para recuperar dados de uma tabela ou exibição. É usada para buscar informações específicas de uma tabela, selecionando determinados campos ou colunas.


2. INSERT:
 é usada para inserir novos dados em uma tabela ou visão. É comumente usado para adicionar registros em uma tabela.


3. UPDATE:
 é usado para modificar dados existentes em uma tabela ou visão. É comumente usado para atualizar um registro em uma tabela.


4. DELETE:
 é usado para remover registros de uma tabela ou visão. É comumente usado para excluir registros obsoletos ou inválidos.


5. JOIN
: é usado para combinar dados de duas ou mais tabelas com base em uma condição de associação.


6. GROUP BY:
 é usado para agrupar dados em uma ou mais colunas e, em seguida, aplicar uma função agregada a cada grupo. É comumente usado para calcular estatísticas, como a média ou a soma.


7. ORDER BY
: é usado para classificar os resultados de uma consulta em ordem crescente ou decrescente com base em uma ou mais colunas.
